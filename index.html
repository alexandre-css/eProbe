Funcionalidades Inteligentes

Desenvolvido especificamente para o sistema eProc/TJSC com foco em produtividade e precisão

Detecção Automática

Identifica automaticamente documentos relevantes (SENT1, INIC1) e extrai dados de sessões das minutas do processo com precisão.

Extração de Texto Avançada

Suporte completo para documentos PDF e HTML com múltiplas estratégias de extração e limpeza inteligente de conteúdo.

Integração com IA

Envio direto para ChatGPT/Perplexity com prefixos otimizados para análise jurídica e processamento inteligente.

Dados de Sessão

Extração automática de datas de julgamento, órgãos julgadores e status das sessões diretamente das minutas.

Minutas Detalhadas

Acesso automatizado aos textos completos de acórdãos e relatórios/votos com interface intuitiva e expansível.

Temas Personalizáveis

4 temas visuais (Azul, Escuro, Claro, Violeta) para personalizar a experiência no sistema eProc.

Instalação Simples

Configure o eProbe em menos de 5 minutos e comece a automatizar seu trabalho

Download

Baixe o código-fonte do projeto eProbe do repositório GitHub oficial.

Chrome Extensions

Abra o Chrome/Edge, vá para chrome://extensions/ ou edge://extensions/e ative o "Modo desenvolvedor".

Carregar Extensão

Clique em "Carregar sem compactação" e selecione a pasta do eProbe.

Pronto!

Navegue para o eProc e veja o botão "Resumir Documento" integrado na página.
