# Testes e Demonstrações do eProbe

Esta pasta contém todos os arquivos de teste, demonstração e validação da extensão eProbe.

## 📁 Estrutura dos Arquivos

### Testes de Interface

-   `teste-popup-liveserver.html` - Teste completo da interface do popup no Live Server
-   `demonstracao.html` - Demonstração interativa das funcionalidades

### Navegação e Índices

-   `demos.html` - Página de navegação entre diferentes demos
-   `demo/` - Pasta com arquivos de demonstração organizados

## 🎯 Como Usar os Testes

### Live Server Testing

1. Abra `teste-popup-liveserver.html` no Live Server
2. Teste todas as funcionalidades do accordion
3. Verifique temas de interface e botões
4. Valide comportamento dos toggles

### Demonstrações Interativas

1. Acesse `demos.html` para navegar entre demos
2. Use `demonstracao.html` para testes completos
3. Explore a pasta `demo/` para exemplos específicos

## 🔧 Funcionalidades Testáveis

-   ✅ Accordion de configurações
-   ✅ Seleção de temas de interface (4 opções)
-   ✅ Seleção de temas de botões (6 opções)
-   ✅ Toggles de configuração
-   ✅ Footer profissional
-   ✅ Responsividade

## 🚀 Desenvolvimento

Para testar novos recursos:

1. Copie `teste-popup-liveserver.html` como base
2. Modifique conforme necessário
3. Teste no Live Server antes de integrar na extensão
